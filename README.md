# Task Manager — управління проєктами та завданнями

Це односторінковий додаток (SPA) для управління проєктами та завданнями всередині них.  
Розроблений як тестове завдання з використанням сучасного стеку Vue 3.

## Основний функціонал

- Список усіх проєктів з пошуком та сортуванням
- Додавання нового проєкту через модальне вікно
- Детальна сторінка проєкту з двома режимами перегляду:
  - Таблиця завдань (сортування, фільтрація за виконавцем та статусом)
  - Канбан-дошка (drag-and-drop між статусами та всередині колонок)
- Створення / редагування завдань з валідацією
- Збереження всіх даних у localStorage (стан зберігається після перезавантаження)
- Перемикання режиму перегляду (таблиця / канбан) зберігається для кожного проєкту окремо

## Технологічний стек

- **Vue 3** — Composition API + `<script setup>`
- **TypeScript** — повна типізація проєктів, завдань, сторів
- **Pinia** — менеджер стану + плагін `pinia-plugin-persistedstate` для localStorage
- **Vue Router** — навігація між сторінками
- **SCSS** — стилізація (змінні, глобальні стилі)
- **Drag & Drop** — нативний HTML5 Drag & Drop (без додаткових бібліотек)
- **Vite** — інструмент збірки

## Як запустити локально

1. Клонуйте репозиторій

   git clone https://github.com/твій-нік/task-manager.git
   cd task-manager

2. Встановіть залежності

   npm install

3. Запустіть у режимі розробки

   npm run dev

   Додаток буде доступний за адресою: http://localhost:3001

## Обмеження та можливі покращення

- Дані зберігаються тільки в localStorage (немає реального бекенду)
- Немає авторизації / кількох користувачів
- Немає глибокої валідації дат / виконавців (виконавець — простий текст)
- Немає пагінації / великих даних (для тестового завдання не потрібно)

Можливі покращення:

- Додати json-server або mock API
- Реалізувати редагування проєктів
- Додати статистику / графіки (наприклад Chart.js)
- Темна тема
- Мобільна адаптивність канбану
